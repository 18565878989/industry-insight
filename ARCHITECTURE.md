## 5. API 设计

```
┌─────────────────────────────────────────────────────────────────────────────────┐
│                                 API 端点 (Endpoints)                             │
├─────────────────────────────────────────────────────────────────────────────────┤
│                                                                                  │
│  Companies (公司)                                                                │
│  ├── GET    /api/companies              # 获取所有公司                          │
│  ├── GET    /api/companies/{id}         # 获取公司详情                          │
│  ├── POST   /api/companies              # 创建公司                              │
│  ├── PUT    /api/companies/{id}          # 更新公司                              │
│  └── DELETE /api/companies/{id}          # 删除公司                              │
│                                                                                  │
│  Products (产品)                                                                 │
│  ├── GET    /api/products                # 获取所有产品                          │
│  ├── GET    /api/products/{id}           # 获取产品详情                          │
│  ├── POST   /api/products                # 创建产品                              │
│  └── GET    /api/companies/{id}/products # 获取公司的产品                        │
│                                                                                  │
│  Relationships (关系)                                                             │
│  ├── GET    /api/relationships           # 获取所有关系                          │
│  ├── POST   /api/relationships          # 创建关系                              │
│  ├── GET    /api/relationships/{type}   # 按类型获取关系                        │
│  └── GET    /api/companies/{id}/relations# 获取公司的所有关系                    │
│                                                                                  │
│  Analysis (分析)                                                                 │
│  ├── GET    /api/analysis/supply-chain   # 供应链分析                            │
│  ├── GET    /api/analysis/competitors    # 竞争分析                              │
│  ├── GET    /api/analysis/trends         # 趋势分析                              │
│  └── GET    /api/analysis/path?from=X&to=Y# 路径查询                            │
│                                                                                  │
│  Knowledge Graph (知识图谱)                                                      │
│  ├── GET    /api/kg/nodes                # 获取所有节点                          │
│  ├── GET    /api/kg/edges                # 获取所有边                            │
│  ├── GET    /api/kg/graph                # 获取完整图数据                       │
│  └── POST   /api/kg/query                # 复杂图查询                           │
│                                                                                  │
└─────────────────────────────────────────────────────────────────────────────────┘
```

## 7. 核心功能模块

| 模块 | 功能 | 技术实现 |
|------|------|----------|
| **知识图谱** | 实体关系可视化 | D3.js Force-Directed Graph |
| **公司画像** | 公司信息展示 | React Cards |
| **供应链视图** | 上下游关系 | D3.js Sankey Diagram |
| **竞争分析** | 市场份额对比 | ECharts |
| **趋势分析** | 时间序列预测 | Chart.js |
| **路径查询** | 关联路径发现 | GraphQL/BFS |
